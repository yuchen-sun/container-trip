# Windows

在windows上使用docker需要注意：

- Windows 10 64位：专业版，企业版或教育版。
- 家庭版win10需要在WSL 2中安装docker。
- 必须启用Hyper-V和Containers Windows功能,BIOS设置中启用BIOS级硬件虚拟化支持。

[下载链接](https://docs.docker.com/docker-for-windows/install/)
